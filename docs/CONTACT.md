# Contact

Preferred contact path:

- GitHub issues for repository-specific bugs.
- GitHub discussions for architecture ideas.
- Direct contact only when needed for collaboration.

---

## Public profile note

Avoid publishing personal email directly if you want less spam.

Use either:

- GitHub profile links;
- a dedicated contact email;
- LinkedIn only if the URL is real and polished.

Remove placeholder links such as:

```text
linkedin.com/in/yourprofile
youremail@gmail.com
```
